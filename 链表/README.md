# 奇技淫巧
141:快慢指针判断有没有循环,有的话快指针就会追上慢指针

142:快慢指针找环的入口位置,环外长度a,环内长度b, slow指针与fast指针相遇时, 两者走过的距离为slow = s, fast = 2s = s+nb, 所以s = nb, 
那从起点出发走nb+a的距离就能走到环口, 即slow与环口距离为a, 新开一个指针从起点出发, 走到与slow相遇的位置即为环口
